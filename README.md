# 💳 Subscription Manager – Secure Expense Tracking Platform

Subscription Manager is a full-stack application that helps users track and manage recurring expenses. It provides a secure and scalable system to monitor subscriptions, reduce unnecessary spending, and maintain financial visibility.

---

## ✨ Features

- 🔐 Secure user authentication (OAuth2 / OIDC)
- 📊 Track and manage recurring subscriptions  
- ➕ Create, update, and delete subscription entries  
- 📅 Monitor billing cycles and payment schedules  
- ⚡ Real-time data handling with cloud database  

---

## 🧠 How It Works

1. User logs in securely  
2. Authenticated session is established  
3. User manages subscriptions via dashboard  
4. Backend processes requests via REST APIs  
5. Data is stored and retrieved from database  

---

## ⚙️ Tech Stack

- Frontend: React.js  
- Backend: Spring Boot  
- Database: Firebase Firestore  
- Authentication: WSO2 Asgardeo  
- APIs: REST APIs  

---

## 🔐 Authentication

Authentication is handled using **WSO2 Asgardeo**, implementing:

- OAuth2 / OpenID Connect (OIDC)  
- Secure login and session management  
- Protected routes and user-specific data access  

---

## 🚀 Deployment

- Backend services structured for scalable deployment  
- Cloud-based database using Firebase Firestore  
- Authentication managed via Asgardeo
